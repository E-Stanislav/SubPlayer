# Структура проекта SubPlayer

```
src/
├── main/              # Главный процесс Electron
│   ├── main.js        # Точка входа, создание окна, IPC обработчики
│   └── preload.js     # Безопасный мост между main и renderer процессами
│
├── renderer/          # Процесс рендеринга (UI)
│   ├── index.html     # HTML интерфейс приложения
│   ├── renderer.js    # Логика интерфейса и обработка событий
│   └── styles.css     # Стили приложения
│
└── utils/             # Утилиты и вспомогательные функции
    └── subtitle-utils.js  # Функции для работы с форматом SRT
```

## Описание папок

### `main/`
Содержит код главного процесса Electron. Этот процесс управляет жизненным циклом приложения, создает окна и обрабатывает системные события.

- **main.js** - основной файл главного процесса
  - Создание и управление окнами
  - IPC обработчики для файловых операций
  - Диалоги открытия/сохранения файлов

- **preload.js** - скрипт, выполняющийся перед загрузкой веб-страницы
  - Предоставляет безопасный API для renderer процесса
  - Использует contextBridge для изоляции контекста

### `renderer/`
Содержит код процесса рендеринга - пользовательский интерфейс приложения.

- **index.html** - структура интерфейса
- **renderer.js** - логика работы с DOM, обработка событий пользователя
- **styles.css** - стили и оформление интерфейса

### `utils/`
Вспомогательные функции, используемые в разных частях приложения.

- **subtitle-utils.js** - утилиты для работы с субтитрами
  - Парсинг SRT формата
  - Генерация SRT файлов
  - Форматирование времени
