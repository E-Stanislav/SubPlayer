# SubPlayer (Go) — кроссплатформенный видеопроигрыватель

Минимальный GUI-проигрыватель на **Go** с кроссплатформенным UI (Fyne) и воспроизведением через **mpv**.

## Требования

- **Go 1.22+**
- **mpv** в `PATH`

### Установка mpv

- **macOS (Homebrew)**: `brew install mpv`
- **Ubuntu/Debian**: `sudo apt install mpv`
- **Windows**: поставь mpv (например, portable build), и добавь папку с `mpv.exe` в `PATH`

## Запуск

```bash
go mod tidy
go run .
```

## Что умеет

- Open (выбор видеофайла)
- Play/Pause, Stop
- Seek по таймлайну и кнопками ±10 секунд
- Volume

Видео рендерится в отдельном окне `mpv` (это самый надёжный кроссплатформенный вариант без тяжёлых нативных биндингов).

# Видеопроигрыватель на Python

Простой и функциональный видеопроигрыватель с графическим интерфейсом, созданный с использованием Python, tkinter и OpenCV.

## Возможности

- ✅ Воспроизведение видеофайлов (MP4, AVI, MOV, MKV, FLV, WMV, WEBM)
- ✅ Управление воспроизведением (Play/Pause/Stop)
- ✅ Ползунок для перемотки видео
- ✅ Отображение текущего времени и общей длительности
- ✅ Автоматическое масштабирование видео под размер окна

## Установка и запуск

### Быстрый запуск (рекомендуется)

Используйте скрипт `run.sh` для автоматической установки зависимостей в виртуальное окружение и запуска:

```bash
./run.sh
```

Скрипт автоматически:
- Найдет Python с поддержкой tkinter (включая системный Python на macOS)
- Создаст или пересоздаст виртуальное окружение при необходимости
- Установит все зависимости
- Запустит приложение

### Ручная установка

1. Создайте виртуальное окружение (опционально):
```bash
python3 -m venv venv
source venv/bin/activate  # На Windows: venv\Scripts\activate
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Запустите приложение:
```bash
python video_player.py
```

## Использование

1. Нажмите кнопку "Открыть файл" для выбора видеофайла
2. Используйте кнопки управления для воспроизведения:
   - ▶ - Воспроизведение/Пауза
   - ⏹ - Остановка
3. Используйте ползунок для перемотки видео
4. Время воспроизведения отображается в формате MM:SS

## Требования

- Python 3.7+
- tkinter (встроен в большинство установок Python)
- OpenCV (opencv-python)
- Pillow (PIL)
- NumPy

## Решение проблем

### Ошибка "No module named '_tkinter'" на macOS

Если вы видите эту ошибку, это означает, что ваш Python не имеет поддержки tkinter. Скрипт `run.sh` автоматически попытается найти Python с поддержкой tkinter, включая системный Python.

Если автоматический поиск не помог, попробуйте:

**Вариант 1: Установить Python с tkinter через Homebrew**
```bash
brew install python-tk
./run.sh
```

**Вариант 2: Использовать системный Python вручную**
```bash
# Скрипт автоматически найдет системный Python, но если нужно вручную:
rm -rf venv
/usr/bin/python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python video_player.py
```

**Вариант 3: Установить Python с официального сайта**
Скачайте и установите Python с [python.org](https://www.python.org/downloads/) - он включает tkinter по умолчанию.
