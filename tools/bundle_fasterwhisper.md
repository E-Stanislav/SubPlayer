# Инструкция по использованию faster-whisper

## Преимущества faster-whisper

- **Быстрее**: до 4x быстрее оригинального Whisper
- **Меньше памяти**: оптимизированное использование памяти
- **Python библиотека**: не требует внешних бинарников
- **GPU поддержка**: автоматическое использование CUDA/ROCm если доступно
- **Проще интеграция**: чистый Python API

## Установка

faster-whisper устанавливается автоматически через зависимости проекта:

```bash
pip install -e .
```

## Модели

Модели скачиваются автоматически при первом использовании и кэшируются в `~/.cache/huggingface/`.

Доступные модели:
- `tiny` - самая быстрая, наименьшая точность
- `base` - баланс скорости и точности (рекомендуется)
- `small` - лучше точность
- `medium` - хорошая точность
- `large-v2` - лучшая точность
- `large-v3` - лучшая точность (последняя версия)

## Локальные модели

Если вы хотите использовать локальную модель:

1. Скачайте модель с [Hugging Face](https://huggingface.co/guillaumekln/faster-whisper-{model})
2. Укажите путь к модели в настройках приложения

## GPU ускорение

### CUDA (NVIDIA)

1. Установите CUDA toolkit
2. Установите `faster-whisper` с CUDA поддержкой:
   ```bash
   pip install faster-whisper[cuda]
   ```
3. В настройках выберите устройство: `cuda`

### ROCm (AMD)

1. Установите ROCm
2. Установите `faster-whisper` с ROCm поддержкой
3. В настройках выберите устройство: `cuda` (ROCm использует CUDA API)

## Типы вычислений

- `default` - автоматический выбор
- `int8` - быстрее, меньше памяти, немного хуже точность
- `int8_float16` - баланс для GPU
- `int16` - для CPU
- `float16` - для GPU, хорошая точность

## Для portable сборки

faster-whisper - это Python библиотека, поэтому она автоматически включается в PyInstaller сборку.

**Важно**: Модели не включаются в сборку автоматически. Пользователю нужно будет скачать модель при первом запуске, или вы можете включить модели вручную:

1. Скачайте нужные модели
2. Поместите их в `packaging/models/`
3. В `pyinstaller.spec` добавьте:
   ```python
   datas=[('packaging/models', 'models')],
   ```
4. Обновите код для поиска моделей в папке приложения
